A simple web app that translates text between multiple languages in real time. Built using HTML, CSS, and JavaScript only.

## 📋 Features
- Translate text between 14 languages
- Swap source/target languages instantly
- Copy translated text to clipboard
- Text-to-speech for source and translated text
- Character counter and error handling

## 🛠️ Tech Stack
HTML, CSS, JavaScript (Vanilla) · MyMemory Translation API · Web Speech API

## ▶️ How to Run
Just open `index.html` in any browser — no installation needed.
